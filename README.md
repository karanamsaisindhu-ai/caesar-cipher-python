# caesar-cipher-python


This project implements the classical Caesar Cipher encryption technique using Python. 
The program allows users to encrypt and decrypt messages using a shift key and also 
includes a brute-force option to try all possible shifts.

## Features
- Encrypt plain text messages
- Decrypt cipher text using a shift key
- Brute-force mode to try all possible shifts
- Handles both uppercase and lowercase letters

## Technologies Used
- Python

## How to Run
1. Download or clone the repository
2. Run the program using:

python caesar_cipher.py

3. Choose an option:
   1 - Encrypt
   2 - Decrypt
   3 - Brute Force

## Example
Plain Text: HELLO  
Shift: 3  

Encrypted Text: KHOOR
